# react-autocomplete-component

## Installation
```
npm install @kpajtasev/react-autocomplete-component
```

## Usage
```
import Autocomplete from ""@kpajtasev/react-autocomplete-component"

<Autocomplete onSelect={data={[]}/>
```

## Options format
```
const data = [
    { key: 1, label: "Option 1" },
    ...
];
```
